Описание серверной части arenda-app.ru

ФАЙЛЫ
--------------------
robots.txt                      - для поисковиков
contract.doc                    - образец договора аренды жилья (для начинающих риэлторов)
push.pem                        - сертификат отправки push уведомлений на iOS устройства (можно сгенерировать снова)
404.html                        - страница 404 ошибки (делает редирект на главную страницу сайта)

config.php                      - ОТРЕДАКТИРУЙТЕ настройки (доступ к базе данных, к админ-панели, логин и пароль к смс шлюзу, прочее)

index.php                       - главная страница сайта
feedback.php                    - скрипт отправки формы обратной связи на email с главной страницы сайта
acra.php                        - скрипт для сбора лога по крэшам android приложения (права 777 на папку "android_crashes")
push.php                        - скрипт для работы с push уведомлениями
sms.php                         - скрипт для работы с смс шлюзом websms.ru

v2_reg.php                      - скрипт обработки post запросов при регистрации пользователей
v2_realtor.php                  - скрипт обработки post запросов от риэлторов
v2_user.php                     - скрипт обработки post запросов от соискателей
v2_settings.php                 - скрипт обработки post запросов для сохранения настроек пользователей

mysql_arenda_tables.sql         - импортируйте файл в phpMyAdmin, чтобы создать необходимые таблицы в базе
mysql_backup.sql                - или импортируйте сразу backup всей базы (соискатели, риэлторы, логи и тп)

ПАПКИ
--------------------
agreement                       - в папке пользовательское соглашение лежит
help                            - в папке помощь по включению геолокации (для android устройств)
support                         - редирект на модальное окно обратной связи (на главной странице сайта)
android_crashes                 - в папке хранятся логи крэшей android приложения (не забыть права 777)
appstore                        - редирект на приложение в магазине app store
googleplay                      - редирект на приложение в магазине google play
arenda                          - редирект на приложение в нужном магазине (автоопределение устройства)
img, js                         - картинки и js скрипты для главной страницы сайта
facebook, twitter, vkontakte    - редиректы на паблики в социальных сетях
arenda_admin                    - админ-панель (логин и пароль в файле index.php)
